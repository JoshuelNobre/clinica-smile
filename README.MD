## :information_source: Run Locally

Clone the project

```bash
  git clone https://github.com/JoshuelNobre/clinica-smile.git
```

Go to the project directory

```bash
  cd clinica-smile
```

Build

```bash
  docker build -t clinica-smile .
```

Run

```bash
  docker run -d -p 8080:80 clinica-smile
```

To view page access

```bash
  localhost:8080 
```
